# Product Requirements Document

## 1. 项目概述

Key Train 是一个完全离线运行的儿童键盘指法训练器。当前版本为 **RC1.0**，应用名称为 `仙贝的键盘练习`。

用户双击本地 `index.html` 即可使用，无需安装依赖、无需服务器、无需联网。产品帮助 9-12 岁儿童学习键盘位置、手指分工、Shift 组合键、大小写切换和基础英文输入。

## 2. 目标用户

主要用户：

- 9-12 岁儿童。
- 无计算机经验，或刚开始学习键盘指法。
- 需要短时间、低压力、可离线完成的键盘练习。

辅助用户：

- 家长：希望孩子在无广告、无账号、无联网环境中练习。
- 老师：希望课堂中快速打开、无需配置即可使用。

## 3. 产品目标

- 完全离线运行。
- 双击 `index.html` 即可启动。
- 帮助儿童认识键盘位置和手指分工。
- 帮助儿童从鼠标认识键盘过渡到实体键盘输入。
- 帮助儿童理解 Shift、Caps Lock 和大小写。
- 提供自由随机练习和短文练习。
- 使用 `localStorage` 保存本地进度。
- 保持儿童友好、清晰、无惩罚感。

## 4. 轻量个性化

- 页面主标题：`仙贝的键盘练习`。
- 儿童称呼：`仙贝`。
- 首页鼓励语：`仙贝，今天也来练一会儿吧`。
- “仙贝”只作为本地静态显示文案。
- 不作为账号、登录名、用户 ID 或多用户档案。
- 不引入头像、生日、年龄或其他隐私信息。
- 不上传、不同步、不联网获取或发送任何个性化信息。

## 5. RC1.0 范围

RC1.0 已实现：

- 首页。
- 版本显示：`版本：RC1.0`。

- Caps Lock 讲解。
- 本地进度保存。

## 6. 三大核心模块

### 6.1 入门了解

目标：让儿童用鼠标认识屏幕键盘。

规则：

- 用户点击屏幕键盘。
- 点对才继续。
- 点错不亮色、不进入下一题，只温和提示。
- Shift 组合目标需要先点任意 Shift，再点 baseKey。
- 大小写课程先显示讲解页，再练习 `a A f F j J`。

### 6.2 入门练习

目标：让儿童使用实体键盘输入。

规则：

- 进入练习前显示“把手放好”准备页。
- 用户按实体键盘。
- 按对才继续。
- 按错不进入下一题，只温和提示。
- Shift 组合目标要求输入目标字符。
- 大小写课程区分小写和大写。

### 6.3 随机练习

目标：让儿童练习连续输入。

自由随机练习：

- 基础随机练习。
- 综合随机练习 - 进阶 1。
- 综合随机练习 - 进阶 2。
- 综合随机练习 - 进阶 3。

短文练习：

- `Simple English`：内部 30 篇短文。
- `Numbers and Signs`：内部 30 篇短文。
- 前端只显示两个分组卡片。
- 点击后随机抽取对应分组中的 1 篇。

打字规则：

- 正确字符正常显示。
- 错误字符变红。
- 用户必须删除并修正错误。
- 完成当前组或当前行后按 `Enter` 继续。
- 未完成或有错误时，`Enter` 不继续。

## 7. 安全与隐私

- 不联网。
- 不上传数据。
- 不收集真实姓名、年龄、学校或联系方式。
- 不使用数据库。
- 不使用服务器。
- 本地数据只存储在当前浏览器的 `localStorage`。
- 清理浏览器数据可能删除练习记录。

## 8. 成功标准

RC1.0 应满足：

- 断网状态下可完整使用。
- 双击本地入口文件可启动。
- 三大核心模块均可进入并完成基础流程。
- 入门了解和入门练习共有 27 课。
- 短文练习两个分组各保留 30 篇内部短文。
- 本地进度刷新后仍保留。
- 未生成外部资源依赖。
