# User Flow

## 1. 总览

RC1.0 包含三个核心入口：

- 入门了解。
- 入门练习。
- 随机练习。

目标用户是 9-12 岁、可能没有计算机经验的儿童。页面必须明确告诉用户当前应该用鼠标点击屏幕键盘，还是按实体键盘。

## 2. 启动流程

1. 用户双击 `index.html`。
2. 浏览器打开首页。
3. 首页显示 `仙贝的键盘练习`。
4. 首页显示输入法提醒：`练习前，请先关闭中文输入法，切换到英文输入。`
5. 首页显示入门了解进度、入门练习进度和随机练习完成篇数。
6. 首页显示入口按钮：入门了解、入门练习、随机练习。
7. 首页右侧工具区显示清除本地进度和 `版本：RC1.0`。

不需要登录、联网、安装依赖或选择用户档案。

## 3. 入门了解流程

1. 用户点击 `入门了解`。
2. 页面显示课程选择。
3. 用户选择 27 课中的任意一课。
4. 如果选择第 27 课，先显示大小写讲解页。
5. 进入练习后，页面显示完整屏幕键盘。
6. 页面提示用户用鼠标点击目标键。
7. 点对后当前键变为手指颜色，并进入下一题。
8. 点错后不继续，只显示温和提示。
9. 全部目标完成后显示本轮完成。
10. 用户可以再来一轮、返回课程选择或返回首页。

Shift 组合目标：

1. 页面高亮两个 Shift 和目标 baseKey。
2. 用户先点击任意一个 Shift。
3. 用户再点击 baseKey。
4. 顺序正确才完成目标。

## 4. 入门练习流程

1. 用户点击 `入门练习`。
2. 页面显示课程选择。
3. 用户选择 27 课中的任意一课。
4. 如果选择第 27 课，先显示大小写讲解页。
5. 页面显示“把手放好”准备页。
6. 用户点击 `我放好了`。
7. 页面进入实体键盘输入练习。
8. 按对后进入下一题。
9. 按错后不继续，只显示温和提示。
10. 全部目标完成后保存进度并显示完成反馈。

大小写课程：

- 小写目标必须输入小写。
- 大写目标必须输入大写。
- 使用 Shift 输入大写是推荐方式。
- 如果 Caps Lock 让浏览器输入出正确大写，本阶段也按正确处理。

## 5. 随机练习流程

1. 用户点击 `随机练习`。
2. 页面显示自由随机练习和短文练习。
3. 页面保留输入法提醒。

### 5.1 自由随机练习

入口：

- 基础随机练习。
- 综合随机练习 - 进阶 1。
- 综合随机练习 - 进阶 2。
- 综合随机练习 - 进阶 3。

流程：

1. 用户选择一个入口。
2. 程序生成一组 15-20 个字符的目标文本。
3. 用户在输入框输入。
4. 错误字符变红。
5. 当前组完全正确后显示：`仙贝，这一组完成了。`
6. 用户按 `Enter` 进入下一组。
7. 未完成或有错误时，`Enter` 不进入下一组。

自由随机练习没有终点。

### 5.2 短文练习

前端只显示两个卡片：

- `Simple English`。
- `Numbers and Signs`。

流程：

1. 用户点击 `Simple English` 或 `Numbers and Signs`。
2. 程序从对应分组的 30 篇短文中随机抽取 1 篇。
3. 页面一次只显示当前行。
4. 用户输入当前行。
5. 错误字符变红。
6. 当前行完全正确后显示：`仙贝，这一行完成了。`
7. 用户按 `Enter` 进入下一行。
8. 未完成或有错误时，`Enter` 不进入下一行。
9. 全篇完成后显示：`仙贝，这一篇完成了。`
10. 用户可以再练一篇、返回随机练习或返回首页。

## 6. 本地进度流程

保存内容：

- 入门了解每课完成状态。
- 入门练习每课完成状态。
- 最近一次练习时间。
- 短文练习完成记录。

清除流程：

1. 用户点击 `清除本地进度`。
2. 应用显示确认提示。
3. 用户确认后清除 `keyTrainProgress`。
4. 首页进度回到默认状态。

## 7. 儿童体验检查点

- 每个页面只突出一个主要动作。
- 错误时不惩罚。
- 文案短、具体、温和。
- 打字练习完成后使用 `Enter` 继续，不需要鼠标点击下一行。
- 所有主要流程都有返回首页或返回随机练习路径。
